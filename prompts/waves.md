Iran has used the codename "Operation True Promise 4" to describe its volleys of ballistic missiles and other munitions launched against Israel, the US, and other targets.

> **Note:** The canonical data format is now `data/waves.json` (nested JSON). This CSV schema is retained as the LLM extraction prompt. New data is extracted as CSV then converted via `scripts/csv_to_json.py`. See `docs/data-dictionary.md` for the full JSON field reference including US base targeting columns.

Your task is to return a CSV array containing the following data for each wave so far:

## Column Definitions

- **wave_number** — Sequential integer identifying the wave (1, 2, 3, etc.)
- **wave_codename_farsi** — The Farsi/Persian codename assigned to this specific wave by Iranian military, if any (e.g. `وعده صادق`)
- **wave_codename_english** — English transliteration/translation of the Farsi codename, if any
- **announced_utc** — UTC timestamp of when the wave was publicly announced (ISO 8601 format, e.g. `2026-03-03T14:30:00Z`)
- **announcement_source** — Who made the announcement (e.g. "IRGC", "Iranian state TV (IRIB)", "Sepah News", "Tasnim News Agency")
- **announcement_x_url** — URL to the X/Twitter post announcing or reporting the wave
- **probable_launch_time** — Best estimate of the actual launch time in UTC (ISO 8601 format). May differ from announcement time.
- **launch_time_israel** — Local time in Israel (Asia/Jerusalem) of the probable launch (ISO 8601 with offset, e.g. `2026-02-28T20:00:00+0200`)
- **launch_time_iran** — Local time in Iran (Asia/Tehran) of the probable launch (ISO 8601 with offset, e.g. `2026-02-28T21:30:00+0330`)
- **solar_phase_launch_site** — Integer 0–5 indicating daylight conditions at the launch site at launch time. Scale: 0 = Night (sun below −18°), 1 = Astronomical twilight (−18° to −12°), 2 = Nautical twilight (−12° to −6°), 3 = Civil twilight (−6° to 0°), 4 = Low sun/golden hour (0° to 6°), 5 = Full daylight (above 6°).
- **solar_phase_target** — Integer 0–5 indicating daylight conditions at the primary target at launch time. Same scale as solar_phase_launch_site.
- **conflict_day** — Day of the conflict, with the first day of launches (Feb 28, 2026 in Israel local time) = Day 1.
- **hours_since_last_wave** — Hours elapsed since the previous wave's probable launch time (decimal, e.g. `2.5`). Empty for wave 1.
- **cumulative_munitions** — Running total of estimated munitions launched across all waves up to and including this one. Waves with unknown counts are skipped (total carries forward). Empty if no counts known yet.
- **new_country_targeted** — Boolean (`TRUE`/`FALSE`). Whether this wave's landings_countries includes a country code not seen in any prior wave (escalation indicator).
- **new_weapon_first_use** — Boolean (`TRUE`/`FALSE`). Whether any weapon type (emad, ghadr, sejjil, kheibar_shekan, fattah, shahed_136, shahed_238, cruise_missiles) appears as TRUE for the first time in this wave (capability escalation indicator).
- **payload** — Free-text description of the munitions fired in this wave (e.g. "Emad ballistic missiles and Shahed-136 drones")
- **israel_targeted** — Boolean (`TRUE`/`FALSE`). Whether Israel was a target in this wave.
- **us_bases_targeted** — Boolean (`TRUE`/`FALSE`). Whether US military bases were targeted in this wave.
- **drones_used** — Boolean (`TRUE`/`FALSE`). Whether drones/UAVs were part of the payload.
- **ballistic_missiles_used** — Boolean (`TRUE`/`FALSE`). Whether ballistic missiles were part of the payload.
- **emad_used** — Boolean (`TRUE`/`FALSE`). Whether Emad missiles were used.
- **ghadr_used** — Boolean (`TRUE`/`FALSE`). Whether Ghadr missiles were used.
- **sejjil_used** — Boolean (`TRUE`/`FALSE`). Whether Sejjil missiles were used.
- **kheibar_shekan_used** — Boolean (`TRUE`/`FALSE`). Whether Kheibar Shekan missiles were used.
- **fattah_used** — Boolean (`TRUE`/`FALSE`). Whether Fattah hypersonic missiles were used.
- **shahed_136_used** — Boolean (`TRUE`/`FALSE`). Whether Shahed-136 drones were used.
- **shahed_238_used** — Boolean (`TRUE`/`FALSE`). Whether Shahed-238 jet-powered drones were used.
- **cruise_missiles_used** — Boolean (`TRUE`/`FALSE`). Whether cruise missiles (e.g. Paveh, Hoveyzeh) were used.
- **bm_liquid_fueled** — Boolean (`TRUE`/`FALSE`). Whether liquid-fueled ballistic missiles were used (Emad, Ghadr).
- **bm_solid_fueled** — Boolean (`TRUE`/`FALSE`). Whether solid-fueled ballistic missiles were used (Sejjil, Kheibar Shekan, Fattah).
- **bm_marv_equipped** — Boolean (`TRUE`/`FALSE`). Whether MaRV-equipped (maneuverable reentry vehicle) ballistic missiles were used (Kheibar Shekan, Fattah).
- **bm_hypersonic** — Boolean (`TRUE`/`FALSE`). Whether hypersonic ballistic missiles were used (Fattah).
- **damage** — Free-text description of reported damage from this wave (e.g. "Hits reported on Nevatim airbase runway", or empty if unknown)
- **fatalities** — Integer count of confirmed fatalities resulting from this wave (0 if none confirmed)
- **landings_countries** — Comma-separated list of ISO 3166-1 alpha-2 country codes where munitions landed (e.g. "IL, IQ, JO")
- **targets** — Free-text description of the intended targets (e.g. "Nevatim airbase, Tel Aviv metropolitan area")
- **description** — General free-text summary of the wave
- **launch_site_description** — Free-text description of the reported or estimated launch site(s) (e.g. "Western Iran, near Tabriz")
- **launch_site_lat** — Latitude of the estimated launch site (decimal degrees, e.g. `38.0962`). Leave empty if unknown.
- **launch_site_lon** — Longitude of the estimated launch site (decimal degrees, e.g. `46.2738`). Leave empty if unknown.
- **intercepted** — Boolean (`TRUE`/`FALSE`). Whether any munitions from this wave were reported intercepted by air defense systems.
- **interception_systems** — Comma-separated list of interception systems used (e.g. "Arrow-2, Arrow-3, David's Sling, Iron Dome, THAAD, Patriot, SM-3"). Leave empty if unknown.
- **intercepted_by_israel** — Boolean (`TRUE`/`FALSE`). Whether Israel's own air defenses intercepted munitions in this wave.
- **intercepted_by_us** — Boolean (`TRUE`/`FALSE`). Whether US forces intercepted munitions in this wave.
- **intercepted_by_uk** — Boolean (`TRUE`/`FALSE`). Whether UK forces intercepted munitions in this wave.
- **intercepted_by_jordan** — Boolean (`TRUE`/`FALSE`). Whether Jordanian forces intercepted munitions in this wave.
- **intercepted_by_other** — Free-text listing any other countries that participated in interception (e.g. "France, Saudi Arabia"). Leave empty if none.
- **estimated_intercept_count** — Estimated number of munitions intercepted in this wave. Leave empty if unknown.
- **estimated_intercept_rate** — Estimated interception success rate as a percentage (e.g. `95`). Leave empty if unknown.
- **exoatmospheric_interception** — Boolean (`TRUE`/`FALSE`). Whether any interceptions occurred in the exoatmospheric phase (outside the atmosphere, e.g. Arrow-3, SM-3).
- **endoatmospheric_interception** — Boolean (`TRUE`/`FALSE`). Whether any interceptions occurred in the endoatmospheric phase (within the atmosphere, e.g. Arrow-2, David's Sling, Patriot, Iron Dome).
- **interception_report** — Free-text narrative of the interception effort: who fired what, from where, what was intercepted at what altitude/phase, and any notable failures or successes. Leave empty if unknown.
- **proxy_involvement** — Boolean (`TRUE`/`FALSE`). Whether Hezbollah, Houthis, or other Iranian proxies participated in this wave.
- **proxy_description** — Free-text description of which proxies were involved and their role, if any.
- **wave_duration_minutes** — Estimated duration of the wave in minutes (from first launch to last impact). Leave empty if unknown.
- **estimated_munitions_count** — Estimated total number of munitions launched in this wave. Leave empty if unknown.
- **munitions_targeting_israel** — Estimated number of munitions directed at Israel. Leave empty if unknown.
- **munitions_targeting_us_bases** — Estimated number of munitions directed at US military bases. Leave empty if unknown.
- **civilian_casualties** — Integer count of confirmed civilian fatalities (subset of fatalities). Leave empty if breakdown unknown.
- **military_casualties** — Integer count of confirmed military fatalities (subset of fatalities). Leave empty if breakdown unknown.
- **injuries** — Integer count of confirmed injuries from this wave. Leave empty if unknown.
- **targeted_tel_aviv** — Boolean (`TRUE`/`FALSE`). Whether the Tel Aviv metropolitan area was targeted (includes Bnei Brak, Petah Tikva, HaKirya, Tel Nof, Beit Shams, central Israel).
- **targeted_jerusalem** — Boolean (`TRUE`/`FALSE`). Whether Jerusalem or East Jerusalem was targeted.
- **targeted_haifa** — Boolean (`TRUE`/`FALSE`). Whether the Haifa area was targeted.
- **targeted_negev_beersheba** — Boolean (`TRUE`/`FALSE`). Whether the Negev region was targeted (includes Beersheba, Nevatim airbase).
- **targeted_northern_periphery** — Boolean (`TRUE`/`FALSE`). Whether the northern periphery was targeted (Galilee, Ramat David, northern Israel).
- **targeted_eilat** — Boolean (`TRUE`/`FALSE`). Whether Eilat was targeted.
- **target_lat** — Latitude of the primary target (decimal degrees). Leave empty if unknown.
- **target_lon** — Longitude of the primary target (decimal degrees). Leave empty if unknown.
- **time_between_waves_minutes** — Minutes elapsed since the previous wave ended. Leave empty for wave 1 or if unknown.
- **idf_statement** — Free-text summary of any IDF/Israeli military response or statement regarding this wave. Leave empty if none.
- **sources** — Comma-separated URLs of key sources documenting this wave.
